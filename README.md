# KiCad Projects

## Torpedo-III

A clone of ECP audios Torpedo-III amplifier with a few changes:
- Off board PS
- On board output CCS
- Cascade tube CCS
- Constant current heater

## T-GND

A PCB for combining high and low current grounds correctly.

A clone of the PCB from the Wolverine diyAudio thread. Original done by stuartmp

## GND-LIFT

A PCB for lifting circuit ground from chassis ground.

Also a clone of a PCB from the Wolverine diyAudio thread.
